# 🚀 AI-Powered Civic Issue Intelligence Platform

An intelligent web-based system designed to streamline public service complaint management using AI-driven insights.

---

## 📌 Overview

This platform enables citizens to report civic issues such as electricity outages, water supply problems, internet disruptions, and public transport issues.  
The system uses an intelligent AI engine to automatically classify complaints, assign priority levels, and generate summaries, helping authorities respond faster and more efficiently.

---

## ✨ Key Features

- 🔐 OTP-based secure login system  
- 📍 Location detection using PIN code  
- 📝 Easy complaint submission  
- 🤖 AI-based category detection  
- ⚡ Priority assignment (High / Medium / Low)  
- 🧠 AI-generated complaint summary  
- 📊 Admin dashboard with analytics  
- 🏙 City-wise complaint tracking  
- 📬 Email notifications (OTP, updates)  
- ⭐ User scoring system (reward & penalty)  
- 🚫 Daily complaint limit to prevent misuse  

---

## 🤖 AI Functionality

The system uses a hybrid AI approach:

- Analyzes complaint text (English + Hinglish)  
- Automatically categorizes issues  
- Assigns urgency based on context  
- Generates concise summaries  
- Ensures reliability using fallback logic  

**Example:**

Input:  
`bijli nahi aa rahi kal se urgent hai`

Output:  
- Category: Electricity  
- Priority: High  
- Summary: Electricity issue reported. Immediate attention required.

## 🏗️ System Architecture

- **Frontend:** HTML, CSS, JavaScript  
- **Backend:** PHP  
- **Database:** MySQL  
- **Email Service:** PHPMailer  
- **AI Engine:** Intelligent processing system (Hybrid AI with optional Gemini integration)

## ⚙️ How to Run (Local Setup)

1. Install a PHP server environment  
2. Place project files in server directory  
3. Import the database (.sql file)  
4. Configure database connection in `db.php`  
5. Run the project in browser
